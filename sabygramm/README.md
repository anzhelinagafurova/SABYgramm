### `npm install`
Устанавливает на локальный компьютер все необходимые пакеты для запуска проекта. В папке с проектов должна появиться папка node_modules. Это может занять довольно много времени, не пугайтесь. 

### `npm start`

Запускает проект по адресу:
[http://localhost:3000](http://localhost:3000)
